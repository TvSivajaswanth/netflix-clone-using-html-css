# 🎬 Netflix Clone (HTML & CSS)

A simple and responsive Netflix homepage clone built using **HTML5** and **CSS3**.  
This project replicates the look and feel of Netflix's landing page for learning and practice purposes.

---

## 📌 Features
- 🎥 Netflix-style UI design  
- 📱 Fully responsive layout  
- 🎨 Styled using pure CSS  
- 🖼️ Hero banner section  
- 📂 Movie rows layout  
- 🔤 Clean typography  

---

## 🛠️ Technologies Used
- HTML5  
- CSS3
